**Function 1: get_trainer_analytics(p_trainer_id, p_start, p_end, p_bucket)**
- Returns jsonb with shape: `{ metrics: {...}, trend: [...] }`
- SECURITY INVOKER (trainer's own RLS applies)

**Function 2: get_trainer_peak_hours(p_trainer_id, p_start, p_end)**

- Filter: b.trainer_id = p_trainer_id AND b.status = 'completed' AND b.created_at BETWEEN p_start AND p_end
- No auth.uid() ownership check needed — trainer_id WHERE clause provides scoping

Create `src/lib/analytics.ts` with four exports. This module is pure TypeScript — no React imports, no supabase import. Testable in isolation.

**Export 1: TypeRange type**
```typescript
export type TimeRange = 'week' | 'month' | 'quarter' | 'year';
```

**Export 2: getDateBounds(range: TimeRange): { start: string; end: string }**
- Returns ISO strings for p_start / p_end parameters to supabase.rpc()
- week: 7 days ago → now
- month: 1 month ago → now (setMonth(month - 1))
- quarter: 3 months ago → now (setMonth(month - 3))
- year: 1 year ago → now (setFullYear(year - 1))
- Both start and end use .toISOString()

**Export 3: getBucketParam(range: TimeRange): 'day' | 'week' | 'month'**
- week → 'day' (7 data points)
- month → 'day' (30 data points)
- quarter → 'week' (13 data points)
- year → 'month' (12 data points)

**Export 4: formatBucketLabel(bucket: string, range: TimeRange): string**
- Converts ISO bucket timestamp to a human-readable label
- For 'day' buckets: format as "Mar 14" (month abbr + day)
- For 'week' buckets: format as "Mar 9" (start of week)
- For 'month' buckets: format as "Mar 2026"
- Use `new Date(bucket).toLocaleDateString('en-US', {...})` — no date library needed

**Export 5: EarningRow interface (for CSV type safety)**
```typescript
export interface EarningRow {
  date: string;
  client: string;
  gross: number;
  net: number;
  status: string;
}
```

**Export 6: exportEarningsCSV(rows: EarningRow[], range: TimeRange): void**
- Header: `Date,Client,Gross,Net,Status`
- All fields wrapped in double quotes; embedded quotes escaped as `""` (RFC 4180)
- BOM prefix: `'\uFEFF'` for Excel UTF-8 compatibility
- Filename: `fitrush-earnings-${range}-${new Date().toISOString().slice(0, 10)}.csv`
- Pattern: create Blob → URL.createObjectURL → append anchor → click → removeChild → URL.revokeObjectURL
