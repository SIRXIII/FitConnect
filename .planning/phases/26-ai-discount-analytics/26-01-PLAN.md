    **Step 1: Create the SQL migration** at `Cenlar demand gt 1-17/supabase/migrations/20260319500000_slot_optimization_rpcs.sql`.

    Create two functions:

    `get_trainer_idle_heatmap(p_trainer_id uuid, p_start timestamptz, p_end timestamptz)`:
    - RETURNS TABLE(day_of_week int, hour int, total_count bigint, booked_count bigint)
    - LANGUAGE sql, SECURITY INVOKER
    - SELECT EXTRACT(DOW FROM s.start_time)::int, EXTRACT(HOUR FROM s.start_time)::int, COUNT(*), COUNT(b.id)
    - FROM public.availability_slots s LEFT JOIN public.bookings b ON b.slot_id = s.id AND b.status IN ('confirmed', 'completed')
    - WHERE s.trainer_id = p_trainer_id AND s.start_time BETWEEN p_start AND p_end AND s.deleted_at IS NULL
    - GROUP BY day_of_week, hour ORDER BY day_of_week, hour

    `get_trainer_slot_utilization(p_trainer_id uuid, p_start timestamptz, p_end timestamptz)`:
    - RETURNS jsonb
    - LANGUAGE sql, SECURITY INVOKER
    - Returns jsonb_build_object('total_count', COUNT(*), 'booked_count', COUNT(b.id))
    - Same FROM/JOIN/WHERE as above but no GROUP BY

    **Step 3: Create library** at `Cenlar demand gt 1-17/src/lib/slotOptimization.ts`.

    Export the following interface and functions:

    ```typescript
    export interface IdleHeatmapRow {
      day_of_week: number; // 0-6 (Sun-Sat)
      hour: number;        // 0-23
      total_count: number;
      booked_count: number;
    }

    export interface DiscountRecommendation {
      day_of_week: number;
      hour: number;
      idle_count: number;
      fill_rate_pct: number;  // 0-100
      suggested_discount_min: number;
      suggested_discount_max: number;
    }

    export interface IdleCell {
      total_count: number;
      booked_count: number;
      idle_count: number;
      idle_intensity: number; // 0-1
    }
    ```

    `computeDiscountRecommendations(rows: IdleHeatmapRow[], topN = 5): DiscountRecommendation[]`:
    - Filter rows where total_count > 0
    - Compute idle_count = total_count - booked_count, fill_rate_pct = Math.round((booked_count / total_count) * 100)
    - Discount tiers: fill_rate < 20 -> min 25, fill_rate < 50 -> min 20, else min 10. max = min + 10.
    - Filter idle_count > 0, sort by idle_count desc, slice topN

    `computeOptimizationScore(bookedCount: number, totalCount: number): number`:
    - Guard: if totalCount === 0 return 0
    - Return Math.round((bookedCount / totalCount) * 100)

    `buildIdleCellMap(rows: IdleHeatmapRow[]): Map<string, IdleCell>`:
    - Key format: `${day_of_week}-${hour}`
    - For each row: idle_count = total - booked, idle_intensity = total > 0 ? idle / total : 0
