**2. Create supabase/functions/_shared/gcal-helpers.ts:**
- Import createClient from supabase-js, requireEnv from env.ts
- `getValidAccessToken(connection: { trainer_id: string; access_token: string; refresh_token: string; expires_at: string }, adminClient: any): Promise<string>` — checks if expires_at > now(), if valid returns access_token, otherwise refreshes via POST to https://oauth2.googleapis.com/token with refresh_token grant_type. On success, updates google_calendar_connections row with new access_token and expires_at. On `invalid_grant` error, sets is_active=false and disconnected_reason='token_revoked', throws Error('gcal_token_revoked').
- `withRetry<T>(fn: () => Promise<T>, maxAttempts = 3): Promise<T>` — for loop 0..maxAttempts-1, try fn(), catch: if last attempt rethrow, else await setTimeout with delay = 2^attempt * 1000.
- `createGcalEvent(accessToken: string, event: { summary: string; description: string; location?: string; start: string; end: string }): Promise<string>` — POST to https://www.googleapis.com/calendar/v3/calendars/primary/events with Authorization Bearer, body with summary/description/location/start.dateTime/end.dateTime, extendedProperties.private.source = 'fitrush'. Returns event.id. Throws on non-ok response.
- `deleteGcalEvent(accessToken: string, eventId: string): Promise<void>` — DELETE to https://www.googleapis.com/calendar/v3/calendars/primary/events/{eventId}. Ignores 404/410 (event already gone). Throws on other errors.
- `listGcalEvents(accessToken: string, timeMin: string, timeMax: string): Promise<Array<{ id: string; summary?: string; start: { dateTime?: string }; end: { dateTime?: string } }>>` — GET events with timeMin, timeMax, singleEvents=true, orderBy=startTime, privateExtendedProperty filter to EXCLUDE source=fitrush events. Use maxResults=250. Returns items array.
