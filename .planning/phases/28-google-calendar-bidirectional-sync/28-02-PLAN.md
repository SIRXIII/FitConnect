**1. Create supabase/functions/google-calendar-connect/index.ts:**

Edge Function handling two actions via POST body `{ action: 'connect' | 'disconnect', code?: string }`:

**connect action:**
- Authenticate caller via Authorization header (same pattern as cancel-booking)
- Get trainer_profiles row for the user: `adminClient.from('trainer_profiles').select('id').eq('user_id', user.id).single()`
- Exchange authorization code for tokens: POST to `https://oauth2.googleapis.com/token` with body: code, client_id (from GOOGLE_CLIENT_ID env), client_secret (from GOOGLE_CLIENT_SECRET env), redirect_uri (from GOOGLE_REDIRECT_URI env), grant_type='authorization_code'
- If token exchange fails, return 400 with error message
- Upsert into google_calendar_connections: trainer_id, access_token, refresh_token, expires_at (computed as now + expires_in seconds), is_active=true, disconnected_reason=null, connected_at=now(), last_sync_at=null. Use onConflict: 'trainer_id'
- CRITICAL: Include `prompt: 'consent'` note in code comments to remind frontend must use it (Pitfall 1)
- Return 200 with { success: true, connected_at: ISO string }

**disconnect action:**
- Authenticate caller, get trainer profile
- Delete row from google_calendar_connections where trainer_id matches
- Also delete all gcal_blocked_slots for that trainer
- Also clear gcal_event_id on all bookings for that trainer: `adminClient.from('bookings').update({ gcal_event_id: null }).eq('trainer_id', trainerProfile.id)`
- Return 200 with { success: true }

Standard CORS handling (OPTIONS preflight returns 'ok'). Method must be POST.

**2. Create src/hooks/useGcalConnection.ts:**

Custom hook returning `{ connection, loading, connecting, disconnecting, connect, disconnect, refetch }`:

- `connection`: GcalConnection | null — fetched from `(supabase as any).from('google_calendar_connections').select('*').single()` (use as any per project convention for unregenerated types). Only fetches if trainerProfileId is provided.
- `loading`: boolean — true while initial fetch is in progress
- `connecting`: boolean — true during OAuth exchange
- `disconnecting`: boolean — true during disconnect
- `connect(code: string)`: invokes `supabase.functions.invoke('google-calendar-connect', { body: { action: 'connect', code } })`, then refetches connection
- `disconnect()`: invokes `supabase.functions.invoke('google-calendar-connect', { body: { action: 'disconnect' } })`, then sets connection to null
- `refetch()`: re-queries google_calendar_connections
- useEffect on mount to fetch connection state
- Parameter: `trainerProfileId: string | undefined`
