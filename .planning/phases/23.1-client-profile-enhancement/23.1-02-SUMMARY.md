## Artifacts Delivered

### `ProfileProgressRing.tsx`
SVG ring with `CIRCUMFERENCE = 2 * Math.PI * 36` (~226), `strokeDashoffset` for progress, Framer Motion entrance animation, missing field hint below ring.

### `HealthConditionsChecklist.tsx`
Grid of 6 condition checkboxes (HEALTH_CONDITIONS import) with active/inactive styling, plus textarea for free-text notes with onBlur save.

### `IntensitySlider.tsx`
3-stop pill selector using INTENSITY_LEVELS with green/amber/red color classes, 60px min-height tap targets, connecting track line.

### `GoalRankPicker.tsx`
Ranked list with ChevronUp/ChevronDown/X buttons, AnimatePresence layout animation, unranked pool grid that disables at capacity-3.

### `ClientPassport.tsx` (refactored)
- Progress ring + title in flex header row
- 3 `<section>` elements: "Personal Info", "Health", "Preferences"
- Personal Info: bio (onBlur), age (onBlur), weight (onBlur), height ft+in (onBlur), fitness level pills (onClick)
- Auto-save via `saveField()` helper with `toast.success('Saved', { duration: 1200, id: 'profile-save' })`
- Avatar upload and `compressImage` fully preserved (CPROFILE-01)
- No "Save Fitness Passport" button
