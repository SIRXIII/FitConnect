**2. Add Realtime slot greying subscription:**

Add a `useEffect` that subscribes to `availability_slots` changes for the current trainer:

```typescript
useEffect(() => {
  if (!trainerProfile?.id) return;

  const slotChannel = supabase
    .channel(`slot-realtime-${trainerProfile.id}`)
    .on('postgres_changes', {
      event: 'UPDATE',
      schema: 'public',
      table: 'availability_slots',
      filter: `trainer_id=eq.${trainerProfile.id}`,
    }, (payload) => {
      if (payload.new.is_booked) {
        setBookedSlotIds(prev => [...new Set([...prev, payload.new.id])]);
      }
    })
    .subscribe();

  return () => {
    supabase.removeChannel(slotChannel);
  };
}, [trainerProfile?.id]);
```
