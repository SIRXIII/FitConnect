    **Migration file** (20260323100000_workout_logs.sql):
    Create two tables:

    1. `workout_logs`: id (uuid default gen_random_uuid() primary key), client_id (uuid references auth.users not null), booking_id (uuid references bookings(id) nullable), logged_at (timestamptz default now() not null), notes (text nullable), created_at (timestamptz default now() not null).
    Indexes: idx_workout_logs_client_id on (client_id), idx_workout_logs_booking_id on (booking_id) where booking_id is not null.

    2. `workout_exercises`: id (uuid default gen_random_uuid() primary key), log_id (uuid references workout_logs(id) on delete cascade not null), exercise_name (text not null), exercise_key (text nullable, null for custom exercises), sort_order (integer not null default 0), sets (jsonb not null default '[]'::jsonb).
    Index: idx_workout_exercises_log_id on (log_id).

    RLS (enable on both tables):
    - workout_logs: Client can SELECT/INSERT/UPDATE/DELETE where client_id = auth.uid().
    - workout_logs: Trainer can SELECT where client_id IN (SELECT b.client_id FROM bookings b JOIN trainer_profiles tp ON tp.id = b.trainer_id WHERE tp.user_id = auth.uid()).
    - workout_exercises: Client can SELECT/INSERT/UPDATE/DELETE where log_id IN (SELECT id FROM workout_logs WHERE client_id = auth.uid()).
    - workout_exercises: Trainer can SELECT where log_id IN (SELECT wl.id FROM workout_logs wl WHERE wl.client_id IN (SELECT b.client_id FROM bookings b JOIN trainer_profiles tp ON tp.id = b.trainer_id WHERE tp.user_id = auth.uid())).
