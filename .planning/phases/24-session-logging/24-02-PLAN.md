**Rendering rules:**
- If `notes` is null/empty AND `exercises` is empty array, render NOTHING (return null) — no expand button shown
- Expand button label: "View session notes" — styled as `text-xs text-ink/40 hover:text-ink/70 cursor-pointer flex items-center gap-1` with `ChevronDown` icon (rotate when expanded)
- Expand/collapse: same AnimatePresence + motion.div pattern (height 0 to auto, opacity 0 to 1, duration 200ms ease-out)

**Notes display (inside expanded section):**
- If notes exist: `<p className="text-sm text-ink leading-relaxed">{notes}</p>`
- If notes is null/empty: skip the notes paragraph

**Exercises display (inside expanded section, below notes):**
- If exercises.length > 0: render list with `className="space-y-1 mt-2"`
- Each exercise row: `<p className="text-sm text-ink/70">{entry.name} — {entry.sets} x {entry.reps}</p>`
- Use the multiplication sign: `\u00D7` (not x) — renders as "Squats — 3 x 12"

4. For each completed booking card (in the past bookings section), after the existing card content and before the review section, render:
```tsx
{booking.status === 'completed' && sessionLogsMap.has(booking.id) && (
  <SessionNotesDisplay
    notes={sessionLogsMap.get(booking.id)!.notes}
    exercises={(sessionLogsMap.get(booking.id)!.exercises as ExerciseEntry[]) ?? []}
    expanded={expandedNotes.has(booking.id)}
    onToggle={() => setExpandedNotes(prev => {
      const next = new Set(prev);
      if (next.has(booking.id)) next.delete(booking.id);
      else next.add(booking.id);
      return next;
    })}
  />
)}
```
